``alignparse`` documentation
===================================================

``alignparse`` is a Python package written by `the Bloom lab <https://research.fhcrc.org/bloom/en.html>`_.
It is designed to align long sequencing reads (such as those from PacBio circular consensus sequencing) to targets, and then filter these alignments based on user-provided specifications and parse out sequences or mutations.

See below for information and examples of how to use this package.
